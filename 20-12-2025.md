# Man pages

```shell
man --all --where printf
#
# --- Description ---
/usr/share/man/man1/printf.1.gz
/usr/share/man/man3/printf.3.gz
```

You can also use apropos

```shell
apropos printf
```

## Check if the directory exists, if not create and move into the directory

```shell
cd walter || mkdir -v walter && cd walter
```
